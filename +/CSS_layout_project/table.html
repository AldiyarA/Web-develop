Topics and questions

In this table you will learn about which topics will be covered in our Victorine, as well as about the number of questions for each of them, which will help you prepare in advance.

The Victorine contains questions from the following topics:

  • Mathematics
  • Economics
  • Films and actors
  • History
  • Literature
  • Culture

Questions are categorized by difficulty level as follows:

  1. Very easy - 10 questions - 1 point
  2. Easy - 10 questions - 2 point
  3. Normal - 10 questions - 3 point
  4. Hard - 10 questions - 5 point
  5. Insane - 10 questions - 7 point
  6. Bonus - 1 questions - 15 point
